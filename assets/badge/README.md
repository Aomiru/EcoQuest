# Badge Images

This folder contains badge images for the EcoQuest app.

## Naming Convention

Each badge should have TWO versions:
- `[badge_id].color.png` - Full color version (shown when unlocked)
- `[badge_id].bw.png` - Black & white version (shown when locked)

## Image Specifications

- Format: PNG with transparency
- Recommended size: 512x512 pixels
- Color version: Full color with vibrant colors
- BW version: Grayscale or desaturated version of the color badge

## Creating BW Versions

You can create black & white versions using:
1. Image editing software (Photoshop, GIMP, etc.) - desaturate the color version
2. Online tools - search for "convert image to grayscale"
3. The app will also apply a grayscale filter programmatically as a fallback
